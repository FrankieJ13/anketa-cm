# 📱 Анкета клиента — PWA

## Что это такое?

Ваша анкета упакована как **Progressive Web App (PWA)** — это значит, что её можно установить на телефон или компьютер как обычное приложение, без App Store и Google Play.

---

## 📁 Структура файлов

```
anketa-pwa/
├── index.html       ← Основная анкета (ваш файл + PWA-дополнения)
├── manifest.json    ← Описание приложения для браузера
├── sw.js            ← Service Worker (кэш + работа офлайн)
├── icons/
│   ├── icon-192.png ← Иконка приложения (192×192)
│   └── icon-512.png ← Иконка приложения (512×512)
└── README.md        ← Эта инструкция
```

## 🚀 Как опубликовать (выбери один вариант)

### Вариант 1: GitHub Pages (бесплатно, 5 минут)
1. Создайте репозиторий на [github.com](https://github.com)
2. Загрузите все файлы из этой папки
3. Зайдите в Settings → Pages → выберите ветку `main`
4. Через минуту анкета будет доступна по адресу `https://ваш-логин.github.io/имя-репо/`

### Вариант 2: Netlify (бесплатно, drag & drop)
1. Зайдите на [netlify.com](https://netlify.com)
2. Перетащите папку `anketa-pwa` прямо на сайт
3. Готово! Получите адрес вида `https://random-name.netlify.app`

### Вариант 3: Vercel (бесплатно)
1. Зайдите на [vercel.com](https://vercel.com)
2. Подключите GitHub или перетащите папку
3. Деплой автоматический

## 📲 Как установить приложение

### На Android (Chrome):
1. Откройте анкету в браузере Chrome
2. Нажмите **«Добавить на главный экран»** (появится баннер внизу экрана)
3. Подтвердите установку
4. Иконка появится на рабочем столе

### На iPhone (Safari):
1. Откройте анкету в Safari
2. Нажмите кнопку **«Поделиться»** (квадрат со стрелкой вверх)
3. Выберите **«На экран «Домой»»**
4. Нажмите **«Добавить»**

### На компьютере (Chrome / Edge):
1. Откройте анкету в браузере
2. В адресной строке появится иконка установки (📥)
3. Нажмите «Установить»

## ✅ Что добавлено по сравнению с оригиналом

| Функция | Описание |
|---------|----------|
| **manifest.json** | Браузер знает название, иконку, цвет приложения |
| **Service Worker** | Анкета работает **офлайн** после первого открытия |
| **Баннер установки** | Предлагает установить приложение автоматически |
| **Кэширование CDN** | DaData и html2pdf загружаются быстрее |
| **Apple-метатеги** | Корректная установка на iPhone/iPad |

---

## ⚠️ Важно

- PWA **требует HTTPS**. Все перечисленные хостинги его предоставляют автоматически.
- Открытие через `file://` (двойной клик по файлу) **не активирует** Service Worker — только через http/https.
- Данные формы сохраняются в `localStorage` браузера — они остаются даже после закрытия приложения.
